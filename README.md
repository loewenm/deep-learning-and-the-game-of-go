# deep-learning-and-the-game-of-go
Using deep learning to play the game of go.

This work is heavily based upon the textbook <i>Deep Learning and the Game of Go</i> by Max Puperla and Kevin Ferguson as published in 2019 and used in the University of Toronto course (#3547) Intelligent Agents & Reinforcement Learning.

Please do not copy or distribute, but rather refer to the original author's source material https://github.com/maxpumperla/deep_learning_and_the_game_of_go.
